> git --version (to check git version).

> vi "filename with extension " (to create file)
> echo "code line" > "filename" (te send code without open file)
> git status (to track status of file)
> git clean -f -d (to clean files or directories from staging area)
> git add "file name" (to add specific file)
> git add . (to add all files in directory)
> git commit -m "message" (message for commit)
> git pull (to pull changes from centrol repo into my local repo)
> git push (to push changes made in local repo to central repo)
> git branch (to see current branch)
> ls -la (to see all and hiden files )
> ls -l (to see all files)
> cat "filename" (to read file)
> git rm "filename" (to delete file)
